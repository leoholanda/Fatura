Versão 1.7.0

Associado imprime declaração para IRPF

Na áerea do associado ao consultar suas faturas o próprio associado pode imprimir declaração para irpf.

Adição de serviços no cadastro do Associado

Novos serviços são adicionados ao associado para registro de valores.

Permite remover fatura

Em caso de erro nos valores da fatura gerada é permitido sua remoção com o código de segurança.

Maior segurança

Usuário com acesso ao modulo 'GERENCIAR USUARIO', pode atribuir ou limitar aos outros usuários o acesso de cada módulo.

Versão 1.6.1

Exibe valor do pagamento

Ajuste na funcionalidade para efetuar pagamento. É calculado o valor de imediato, conforme informação do dia no calendário em exibição.

Mudança na área de convênios

Na página inicial os convênios são exibidos agora em card, isso melhora a visualização nos aparelhos móveis

Versão 1.6.0

Mudança formulário beneficiário

Para atender a necessidade dos beneficiários que não tem plano de saúde. As mudanças realizadas foram:

  • Remoção da lista de planos, onde o usuário adicionava cada item referente aos convênios
  • Novo campo 'Tem plano de saúde ?'
  • Se o beneficiário tiver o plano de saúde habilita a opção 'Acomodação'

Nova opção no menu

Nova opção de Email adicionada no menu.
Serve para enviar email para todos os beneficiário.

Versão 1.5.0

Gera fatura individual.

O sistema alerta o usuário em caso de fatura não gerada para o beneficiário selecionado, com isso abre a opção para gerar a fatura do mês atual.

Versão 1.4.2

Informa o valor pago.

Usuário informa o valor pago da fatura Dessa forma se o valor pago for menor que o valor cobrado este resíduo será informado na próxima fatura.

Versão 1.4.1

Usuário informa a data de pagamento da fatura, com isso o sistema consegue recalcular os valores mesmo se esta atividade ocorrer depois do dia do vencimento.

Envia email de confirmação do pagamento.

Beneficiário recebe no seu email a confirmação do pagamento da fatura.
